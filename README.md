# VideoApp

## Technologies Used

- React Native
- Expo
- TypeScript

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/irinosochka/VideoApp.git

2. **Navigate to the project directory**

   ```bash
   cd VideoApp

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the app**

   ```bash
    npx expo start
   ```
